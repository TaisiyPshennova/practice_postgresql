# Практическая работа по PostgreSQL

**Выполнила:** Пшеннова Таисия

---

## Описание

База данных для хранения информации об учреждениях культурно-досугового типа и образовательных учреждениях.  
Работа выполнена в PostgreSQL с использованием DBeaver.

---

## Файлы

- `script-culture.sql` — запросы по культурным учреждениям
- `script-education.sql` — запросы по образовательным учреждениям
- `education_data.sql` — резервная копия схемы с образовательными учреждениями
- `culture_data_backup.sql` — резервная копия схемы с культурными учреждениями

---

## Технологии

PostgreSQL, DBeaver, PostGIS, Git
